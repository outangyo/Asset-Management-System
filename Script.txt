    Docker Command

docker --version	// version
docker info		// information
docker ps		// image ที่รันอยู่
docker images		// list all images
docker compose up -d // รัน docker compose ใน background
docker compose down // หยุด docker compose

// สร้าง SQL server
docker run -e "ACCEPT_EULA=Y" -e "MSSQL_SA_PASSWORD=YOUR_PASSWORD" -p 1433:1433 --name sql_server --hostname sql_server -d mcr.microsoft.com/mssql/server:2022-latest

// stop image หา id โดยใช้คำสั่ง docker ps
docker container stop 59cf2c1afdcb

    Docker Compose Steps
1. บนเครื่องที่สอง:
    * ติดตั้ง Git
    * ติดตั้ง Docker Desktop
    * ติดตั้ง Postman
2. เปิด Terminal/PowerShell
3. รัน git clone [URL ของ GitHub Repository]
    ตัวอย่าง: git clone
https://www.youtube.com/watch?v=0gZ8MXZ4Izg
4. cd เข้าไปในโฟลเดอร์โปรเจค
5. รัน docker-compose up --build
6. รอจนเสร็จ...
7. เปิด Terminal ใหม่ แล้วรัน docker-compose exec web dotnet ef database update
8. เปิด Postman แล้วยิง API เพื่อ Seed Data
9. เปิด Browser แล้วเข้าไปที่ http://localhost:8080

เปิด Terminal ใหม่
รัน docker-compose exec web /root/.dotnet/tools/dotnet-ef database update
